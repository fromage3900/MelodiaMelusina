# NASA reference assets (Material Maker celestial lane)

Bundled **offline** stand-ins and catalog data for the `SG_Celestial` sampling chain. Replace placeholders with full-resolution NASA downloads when desired.

## Files

| File | Role | Recommended source |
|------|------|-------------------|
| `blue_marble_4k.jpg` | Earth albedo / cloud emissive tint | [NASA Visible Earth — Blue Marble](https://visibleearth.nasa.gov/collection/1484/blue-marble) |
| `hubble_carina_4k.jpg` | Nebula emissive color | STScI / Hubble press release imagery (public domain) |
| `moon_lro_elevation_2k.png` | Height perturbation | [JPL/Caltech tmaps](https://maps.jpl.nasa.gov/tmaps/) |
| `mars_mola_2k.png` | Macro roughness variation | NASA MOLA elevation |
| `star_bright.csv` | Bright star RA/Dec/mag | Bright Star Catalog subset |
| `star_mask.png` | Point mask for MM `image` node | Generated by `preprocess_nasa_stars.py` |

## Generate placeholders

```bash
py Tools/MaterialMaker/generate_nasa_placeholders.py
py Tools/MaterialMaker/preprocess_nasa_stars.py
```
