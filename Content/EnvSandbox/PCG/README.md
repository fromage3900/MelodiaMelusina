# EnvSandbox PCG

Do not add graphs at this root. Use:

- `Universal/` — portfolio graphs (`PCG_FoliageDensity`, `PCG_RockScatter`, …)
- `Greybox/` — blockout presets
- `Collections/` — `SMC_*` scatter kits
- `Styles/<Style>/` — per-style kits (Sakura, Baroque)
- `_Deprecated/` — retired orphans (e.g. legacy `PCG_MeadowScatter`)

`_PROJECT/PCG` is read-only Melodia reference. Build and audit via `Content/Python/setup_pcg_universal.py` and `audit_pcg_portfolio.py`.
